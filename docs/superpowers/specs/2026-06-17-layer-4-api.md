## `shell/job_store.py`

In-memory store for tracked jobs.

```python
@dataclass
class TrackedJob:
    job_id: str
    request: JobRequest
    strategy: ExecutorStrategy
    plan: Plan | None = None
    result: PlanResult | None = None
    status: str = "pending"  # "pending" | "running" | "completed" | "failed"

class JobStore:
    def put(self, job: TrackedJob) -> None: ...
    def get(self, job_id: str) -> TrackedJob | None: ...
    def list_all(self) -> tuple[TrackedJob, ...]: ...
```

## `shell/orchestrator.py`

Service layer wiring registry, planner, executors, cache, and job store.

```python
class Orchestrator:
    def __init__(self, registry: WorkerRegistry, cache: PlanCache, data_dir: Path) -> None: ...

    async def submit_job(self, request: JobRequest, strategy: ExecutorStrategy) -> TrackedJob: ...
    async def get_job(self, job_id: str) -> TrackedJob | None: ...
    async def list_jobs(self) -> tuple[TrackedJob, ...]: ...
    def get_capabilities(self, src: str | None, dst: str | None) -> dict: ...
    def register_worker(self, worker_id: str, endpoint: str, transport: str, capabilities: WorkerCapabilities) -> None: ...
    def list_workers(self) -> tuple[RegisteredWorker, ...]: ...
```

- `submit_job`: compiles plan, stores in JobStore, runs executor in background, updates status
- `get_capabilities`: aggregates language pairs from registry

## `shell/api/schemas.py`

Pydantic models for API request/response:

```python
class SubmitJobRequest(BaseModel):
    source_type: str          # "epub" | "audio"
    source_path: str
    source_language: str
    target_language: str
    executor_strategy: str = "batch_async"
    asr_model: str | None = None

class JobResponse(BaseModel):
    job_id: str
    status: str
    plan_id: str | None
    completed_steps: int
    total_steps: int
    errors: list[str]

class WorkerRegistrationRequest(BaseModel):
    worker_id: str
    endpoint: str
    transport: str
    capabilities: WorkerCapabilitiesModel

class CapabilitiesResponse(BaseModel):
    language_pairs: list[LanguagePair]
```

## `shell/api/routes/`

### workers.py

- `GET /workers` — list workers

### capabilities.py
- `GET /capabilities` — aggregate language pairs
